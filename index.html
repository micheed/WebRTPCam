How it works:
The signal server only helps your phone and OBS find each other (exchanges ~1KB of text). After that, video streams directly peer-to-peer on your local network — never through the server.
Deploy signal server (free, 5 min):
1. Go to render.com → New Web Service
2. Create a GitHub repo with server.js + package.json
3. Build: npm install   Start: node server.js
4. Copy your wss://xxx.onrender.com URL
5. Paste it in the Sender tab config box
OBS Browser Source setup:
Sources → Add → Browser Source
Paste the OBS URL from Sender tab
Width: 1920 Height: 1080 (or 3840×2160 for 4K)

4K note:
iPhones cap at 1080p in Safari. Android Chrome may allow 4K on newer devices. App auto-falls back to 1080p if unsupported.
PIN security:
PINs are SHA-256 hashed — never sent in plaintext. Protects your stream on shared networks.
Want true NDI?
Install obs-ndi plugin → enable OBS Virtual Camera → other LAN devices see it as an NDI source.
