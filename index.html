No signaling server needed.
LiveCam uses your local Wi-Fi only. Signaling happens via browser localStorage — no cloud, no accounts, zero cost.
How it works:
1. Phone & OBS on same Wi-Fi
2. Sender taps Start → gets a Room ID
3. OBS Browser Source opens viewer URL
4. WebRTC connects directly — phone ▶ OBS
5. Video stays on your local network only
OBS Browser Source setup:
Sources → Add → Browser Source
Paste the OBS URL from Sender tab
Width: 1920   Height: 1080
For 4K: 3840 x 2160
✓ Shutdown source when not visible
✓ Refresh when scene becomes active
4K note:
iPhones cap at 1080p in Safari via WebRTC. Android Chrome may support 4K on newer devices. App auto-falls back to 1080p if 4K fails.
PIN security:
PIN is SHA-256 hashed before comparison — never stored or sent in plaintext. Protects your stream on shared Wi-Fi networks. Anyone without the PIN sees a lock screen.
Want true NDI from OBS?
Install free obs-ndi plugin on your PC. Enable Virtual Camera — other LAN devices see it as a native NDI source.
