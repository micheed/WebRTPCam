How to stream to OBS in 3 steps:

1. On your phone (Sender tab):
  › Tap START STREAM
  › Copy the OBS URL

2. In OBS on your computer:
  › Add Source → Browser Source
  › Paste the URL
  › Set W:1280 H:720

3. Start your OBS scene
  › Your phone camera appears live!

⚠ Same Wi-Fi required for lowest latency. Works over internet too but adds delay.
Why not NDI?

True NDI requires a native app (SDK). This app uses WebRTC — the same tech used by Google Meet and Zoom — which works natively in OBS via Browser Source with near-zero latency on local Wi-Fi.

For true NDI output: install OBS-NDI plugin on your desktop, then use OBS Virtual Camera to re-broadcast your Browser Source as NDI to other devices.
Tips for best quality:

› Use same Wi-Fi network
› Keep phone plugged in
› Select FHD 1080p for sharpest image
› In OBS, match Browser Source resolution to your scene output
› Disable phone auto-lock while streaming
